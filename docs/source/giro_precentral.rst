giro precentral
===============

El giro precentral es una estructura clave del cerebro ubicada en el lóbulo frontal, anterior al surco central. Su función principal es la corteza motora primaria, que controla los movimientos voluntarios del cuerpo. Esta área es esencial para planificar y ejecutar movimientos y está organizada somatotópicamente, de modo que diferentes partes de esta corteza se encargan de controlar diferentes partes del cuerpo (conocido como homúnculo motor)
